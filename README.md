# Codesoft-Calculator
Interactive Calculator
A fully functional calculator application built with modern web technologies:

    CSS Grid Layout: Professional 4×5 button arrangement

    Complete Functionality: All basic arithmetic operations (+, -, ×, ÷)

    Advanced Features:

        Decimal point support

        Backspace functionality

        Clear (C) and All Clear (AC) buttons

        Sign toggle (±) button

        Keyboard support for input

    Error Handling: Division by zero protection and invalid operation handling
    

Modern Design: Sleek dark interface with gradient buttons and smooth animations
